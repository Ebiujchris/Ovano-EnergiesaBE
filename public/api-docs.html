📦 Products

GET /api/products?userId={userId}
Get all products for a user
POST /api/products
Add a new product
{ "name": "Sugar", "buyingPrice": 3000, "sellingPrice": 3500, "stockQuantity": 50, "category": "Food", "userId": "b5ba2aea-d3f5-41ba-82ea-4da2eeda69d7" }
GET /api/products/low-stock/{userId}
Get products with low stock
GET /api/products/{id}
Get a specific product
PATCH /api/products/{id}
Update a product
DELETE /api/products/{id}
Delete a product

💰 Sales

GET /api/sales?userId={userId}
Get all sales for a user

📊 Dashboard

GET /api/dashboard/{userId}
Get complete dashboard data
Returns: Today's sales, profit, transactions, low stock items, credit stats, and recent activity
GET /api/dashboard/{userId}/analytics?period=month
Get analytics data (period: week, month, year)
